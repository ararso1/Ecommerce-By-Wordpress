== Description ==

The Dokan Order Approval plugin enables vendors to approve or reject customer orders before payment is processed.

**Important Note** 

This plugin will only work if you have the [Order Approval for WooCommerce](https://WordPress.org/plugins/order-approval-WooCommerce/ "dokan order approval for WooCommerce") and [Dokan](https://WordPress.org/plugins/dokan-lite/ "WooCommerce venders order approval") plugin installed and activated.

**Sales process flow for new order**

1. The customer places an order, but no payment is processed at this time.
1. The shop owner/admin/vendor then has the option to approve or reject the order.
1. If the order is approved, the customer will receive a notification email that includes a payment link.
1. If the order is rejected, the customer will receive a cancellation email.

**Features**

1. After installing and activating the plugin, the shop owner/vendor can approve or reject customer orders.
1. Orders must be manually approved by the store owner before payment is processed.
1. The plugin will send a notification email to the customer whenever an order is approved or rejected.
1. If an order is approved, the customer will receive an email with a payment link.
1. This plugin supports all enabled payment gateways.
1. There are no complex settings to configure.
1. Blocked checkout feature.
1. Admin can edit the orders.

== Installation ==

1. Activate the plugin in your WordPress admin area.
1. Open the settings page for WooCommerce and click the “Payment Gateways” tab.
1. Enable “WooCommerce pre-order ” payment gateway.
1. Goto Email Tab and customise emails.

== Frequently Asked Questions ==

= Will it work with all payment gateways such as paypal, stripe etc. =

The plugin dokan order approval for WooCommerce has been designed to work with the default WooCommerce payment gateways. If you are facing any troubles please contact via support.

= Will it take customer card details while ordering? =

No. Customers can make an order without any payment. Later on approval of the order they can make payment.It's the main concept of dokan order approval.vendor approval needs for order plugin

= What will be the order status after making a new order? =

When user make and order it will move to a custom status called "waiting". On approval it will move to "pending".On Rejection it will move to "cancelled"
